Как это работает

Режим Юзербота

Бот авторизуется как ваш личный клиент. Перехватывает всё: текст, фото, файлы, стикеры, кружочки и голосовые сообщения во всех типах чатов.

Самый мощный режим. Требует API ID.

Telegram Business

Использует официальный API. Перехватывает удаленку только в личных чатах, работает на 100% официально без авторизации сторонних сессий.

Требует наличие Telegram Premium.

Умная сортировка

HordaGram автоматически создает скрытые системные группы. Текст идет в одну папку, а медиа (фото, файлы, кружки) — в другую. Полный порядок.

Получение API

Эти ключи необходимы для авторизации вашего Юзербота напрямую через сервера Telegram. Процесс абсолютно безопасен.

1

Официальный портал

Перейдите на сайт для разработчиков: my.telegram.org.

2

Авторизация

Введите свой номер телефона. Системный код подтверждения придет прямо в приложение Telegram.

3

Раздел разработки

После входа выберите пункт меню «API development tools».

4

Создание приложения

Заполните только первые поля формы:

  • App title: Напишите любое слово (например App123)
  • Short name: Любое слово без пробелов (например app123)
  • Platform: Выберите Android или Desktop
5

Завершение

Нажмите синюю кнопку «Create application». Скопируйте появившиеся цифры App api_id и строку App api_hash. Отправьте их боту.

Политика безопасности

Конфиденциальность

Архитектура построена на строгой анонимности:

  • Мы не сохраняем ваши переписки, фото или файлы на своих серверах.
  • Все перехваченные сообщения пересылаются напрямую в ваши личные закрытые группы.
  • Данные API надежно зашифрованы и не передаются 3-м лицам.
  • Только вы имеете доступ к своему боту-ассистенту.

Строго запрещено

Нарушение правил приведет к вечной блокировке:

  • Спам: Использование юзербота для массовой рассылки рекламы.
  • Скам: Использование мощностей в мошеннических схемах.
  • Накрутка: Запуск скриптов для накрутки реакций и просмотров.
  • Нарушение правил: Любые действия, вредящие мессенджеру Telegram.
